# Personal Portfolio Website

This is my personal developer portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
The website showcases my experience, technical skills, and selected projects as a **Fullstack JavaScript Developer**.

## Live Preview
You can deploy and view this project using **Vercel** or any platform that supports Next.js.

## Tech Stack

The main technologies used in this project:

- **Next.js** – React framework for building modern web applications
- **TypeScript** – Strongly typed JavaScript for better maintainability
- **Tailwind CSS** – Utility‑first CSS framework for fast UI development
- **Lucide React** – Modern icon library
- **React** – Component based UI library

## Features

- Responsive portfolio layout
- Modern UI design
- Experience timeline
- Skills highlight section
- Contact section with WhatsApp and Email
- Optimized performance using Next.js

## Project Structure

```
app/
components/
public/
styles/
```

- **app/** – Main Next.js app router pages
- **components/** – Reusable UI components
- **public/** – Static assets such as images
- **styles/** – Global styling

## Getting Started

Clone the repository:

```
git clone https://github.com/your-username/your-repository-name.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```

## Deployment

The easiest way to deploy this project is with **Vercel**.

Steps:

1. Push the repository to GitHub
2. Import the project into Vercel
3. Deploy automatically
